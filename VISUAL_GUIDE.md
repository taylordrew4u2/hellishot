## 🎨 Color Scheme

### Primary Colors
- **Red**: `#DC2626` (buttons, accents, brand)
- **Orange**: `#F97316` ("Next Up" blocks)
- **Green**: `#10B981` (success, available slots)
- **Gray**: Various shades for text and backgrounds

### Status Indicators
- **Available**: Green progress bar
- **Full**: Red progress bar with "FULL" badge
- **Next Up**: Orange border and "NEXT UP" badge
- **Live**: Red border with animated "LIVE" badge
- **Paid**: Green badge
- **Cash Pending**: Yellow badge

## 📱 Mobile Layout

On mobile devices (< 768px):
- Single column layout
- Full-width buttons
- Touch-friendly spacing
- Larger tap targets (44x44px minimum)
- Simplified navigation
- Bottom-sheet style modals

## 💻 Desktop Layout

On desktop (> 1024px):
- Max width container (896px)
- Centered content
- Larger typography
- More whitespace
- Side-by-side stats
- Hover effects on buttons

## 🎭 Animations

### Smooth Transitions
- Modal slide-in/out
- Button hover effects
- Progress bar fill animation
- Loading spinners
- Success checkmark animation
- "LIVE" pulse animation
